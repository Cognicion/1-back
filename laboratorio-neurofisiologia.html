Simulador 1

Membrana neuronal e intercambio ionico

Como usar este simulador
  1. Elige un preset o modifica concentraciones intra/extracelulares para observar como cambia Vm.
  2. Selecciona el ion protagonista para enfocar su gradiente, canal y direccion de flujo.
  3. Usa el tiempo de movimiento para pasar de una vista rapida educativa a camara lenta.
  4. Activa o desactiva iones, cargas, canales y flechas para estudiar cada capa por separado.
  5. Compara Vm con Eion y revisa la tabla para integrar gradiente quimico, electrico y flujo neto.

Las particulas son una representacion pedagogica sincronizada con el modelo; no equivalen al numero real de iones.

Simulador educativo

TEC: campo electrico y respuesta cortical

Modelo didactico de terapia electroconvulsiva. No calcula dosis clinicas reales ni sustituye protocolos institucionales.

Que representa

Estimulo electrico controlado

El simulador muestra como los parametros del tren modifican una estimacion educativa de carga, campo y reclutamiento cortical. Los valores son relativos y no deben usarse para indicacion, dosis ni seguridad clinica.

Secuencia fisiologica simplificada
  1. El dispositivo aplica un tren breve de pulsos.
  2. El campo atraviesa tejidos con impedancia variable.
  3. Se despolarizan redes corticales suficientes si se supera el umbral.
  4. La respuesta se monitoriza clinicamente con EEG, motor y signos vitales.
Advertencia

Modelo de educacion medica. TEC real requiere evaluacion, consentimiento, anestesia, monitorizacion y protocolos especializados.

Nernst

Calcula el potencial de equilibrio de un ion a partir de concentraciones intra/extracelulares, valencia y temperatura.

Goldman-Hodgkin-Katz

Estima Vm ponderando concentraciones por permeabilidad. Para Cl- se invierten las concentraciones por ser anion.

Hodgkin-Huxley reducido

Modelo educativo con compuertas m, h y n para Na+ y K+ que genera fases del potencial de accion.

Mielina

Reduce capacitancia efectiva y favorece conduccion saltatoria entre nodos de Ranvier.
