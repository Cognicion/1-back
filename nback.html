Instrucciones Dual N-Back

Observa la posición del cuadro iluminado y escucha el número. Compara cada estímulo con el que apareció N turnos antes.

Sonido

Presiona A o el botón SONIDO solo cuando el número actual coincida con el de N turnos antes.

Posición

Presiona D o el botón POSICIÓN solo cuando el cuadro actual coincida con el de N turnos antes.

Sin coincidencia

No presiones nada. Esa respuesta cuenta como rechazo correcto de no coincidencia.

Acierto: respondes a una coincidencia. Comisión/falsa alarma: respondes cuando no había coincidencia. Omisión: no respondes a una coincidencia.
