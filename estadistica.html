Estadistica clinica

Analisis descriptivo, inferencial y utilidad diagnostica para investigacion medica.

Datos

Pega una tabla CSV/TSV con encabezados o carga un archivo. Ejemplo: grupo, edad, phq9, respuesta.

Guia para principiantes

El programa detecta el tipo de cada variable y te sugiere el analisis mas util segun lo que selecciones.

Modo guiado
  1. Carga pacientes autorizados o pega una tabla con encabezados.
  2. Revisa los tipos de variable y corrige alguno si el sistema se equivoca.
  3. Elige una variable principal: por ejemplo edad, PHQ-9, diagnostico o respuesta.
  4. Elige una variable X o grupo si quieres comparar dos columnas.
  5. Lee la recomendacion y ejecuta el analisis o grafica sugerida.
  6. Exporta los resultados cuando ya esten listos para tu nota o reporte.
